About the AQI Tracker

The AQI Tracker is a project built for HackNjIT 2024. The AQI Tracker is a web application that provides real-time air quality insights and historical data analysis. Our mission is to allow users to make informed decisions about the air quality in their environment.

Project Features

  • Real-time air quality data access
  • Interactive maps to visualize air quality
  • Historical data analysis to observe trends
  • Customized recommendations based on user inputs

Technology Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python, Flask
  • APIs: OpenWeatherMap
  • Deployment: GitHub

Our Inspiration

We were inspired to create the AQI Tracker to address the growing air pollution crisis. We wanted to make an interactive tool that allows the user to engage and understand air pollution metrics. We hope that the AQI Tracker allows users to gain insights on Air pollution in different regions
